# 📚 Online Bookstore Website

A responsive and user-friendly online bookstore website built using **HTML**, **CSS**, and **JavaScript**. The project provides a clean interface for browsing books, viewing book details, exploring best sellers and new releases, and accessing customer support pages.

## 🚀 Features

- 🏠 Home page
- 📖 Book catalog
- ⭐ Best sellers section
- 🆕 New releases
- 📘 Individual book detail page
- 👥 Book clubs information
- 🎁 Gift cards page
- 📧 Newsletter subscription page
- 📞 Contact page
- ❓ FAQ page
- 🚚 Shipping information
- 🔄 Returns policy
- 🌐 Social media pages (Facebook, Instagram, Twitter)
- 📱 Responsive design
- 🎨 Modern and clean UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
